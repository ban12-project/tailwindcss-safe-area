# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

- Nothing yet!

## [0.3.0] - 2026-04-21

### Added

- Add Tailwind CSS v4 compatibility while keeping Tailwind CSS v3 support.
- Add local and GitHub Actions test matrices for Tailwind CSS v3 and v4.
- Document Tailwind CSS v4 `@plugin` usage.

### Changed

- Simplify safe area utility generation with a shared table-driven implementation.
- Use shorthand CSS output for equivalent all-side margin, padding, scroll-margin, and scroll-padding utilities.
- Update the supported Tailwind CSS peer dependency range to `>=3.2.0 <5`.

## [0.2.4] - 2024-12-27

- Export es6 module

## [0.1.0] - 2023-08-18

### Added

- Add new `safe` utilities.
- Add new `safe-max` variants.
